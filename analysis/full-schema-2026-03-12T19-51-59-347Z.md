1. ADMIN SETUP
   ├── Add ingredient "Harina" → ingredients table
   ├── Add supplier "Harinas del Norte" → suppliers table
   ├── Create recipe "Oreo" → recipes table
   └── Link ingredient to supplier → update ingredients

2. PRODUCTION
   ├── Click "Plan Batch" in calculator
   ├── System creates production_batches record
   ├── System subtracts ingredients → inventory_transactions
   ├── System updates ingredients.current_stock
   └── Quality check creates batch_quality_control

3. SELLING
   ├── Customer orders → creates orders record
   ├── Order items created → order_items table
   ├── If new customer → creates clients record
   └── Links order to client → update orders

4. INVENTORY MANAGEMENT
   ├── Daily check finds low stock
   ├── Creates shopping_list items
   ├── Create purchase_order to supplier
   ├── Update shopping_list status
   └── When ingredients arrive → inventory_transactions + update stock

5. FINANCIALS
   ├── Record expenses (rent, utilities) → expenses table
   ├── Categorize expenses → expense_categories
   └── Track all costs for reporting
